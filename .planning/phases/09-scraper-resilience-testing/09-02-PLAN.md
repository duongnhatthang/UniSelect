    1. Create `tests/fixtures/ocr-smoke-generator.py` — a Python script that generates a tiny synthetic JPEG with Vietnamese text for OCR testing. This script will be run as part of the CI workflow to create the test image on-the-fly (avoids checking binary into git).

       ```python
       #!/usr/bin/env python3
       """Generate a small synthetic JPEG with Vietnamese text for OCR smoke testing."""
       import sys
       from PIL import Image, ImageDraw, ImageFont

       output_path = sys.argv[1] if len(sys.argv) > 1 else 'tests/fixtures/ocr-smoke.jpg'

       img = Image.new('RGB', (400, 200), color='white')
       draw = ImageDraw.Draw(img)
       # Use default font (always available)
       try:
           font = ImageFont.truetype('/usr/share/fonts/truetype/dejavu/DejaVuSans.ttf', 20)
       except (OSError, IOError):
           font = ImageFont.load_default()

       lines = [
           'Ma nganh: 7480201',
           'Diem chuan: 26.50',
           'To hop: A00',
       ]
       y = 20
       for line in lines:
           draw.text((20, y), line, fill='black', font=font)
           y += 40

       img.save(output_path, 'JPEG', quality=95)
       print(f'Generated OCR smoke test image: {output_path}')
       ```

       Note: Uses ASCII-safe Vietnamese (no diacritics) to work with any font. PaddleOCR can still extract the text.

    2. Create `.github/workflows/ci-ocr.yml` with the following structure:

       **Trigger:** on push to `scripts/ocr_table.py` or `.github/workflows/ci-ocr.yml`, plus `workflow_dispatch` for manual runs.

       **Job: ocr-smoke** (runs-on: ubuntu-latest)

       Steps:
       a. `actions/checkout@v4`
       b. `actions/setup-python@v5` with python-version '3.11' and cache 'pip'
       c. `actions/cache@v4` for PaddleOCR models:
          - path: `~/.paddlex`
          - key: `paddleocr-models-${{ runner.os }}-${{ hashFiles('scripts/ocr_table.py') }}`
          - restore-keys: `paddleocr-models-${{ runner.os }}-`
       d. Install dependencies: `pip install paddleocr Pillow`
       e. Set env var `PADDLE_PDX_MODEL_SOURCE=BOS` (safety net for offline model use after cache restore)
       f. Warm up / download PaddleOCR models: `python3 -c "from paddleocr import PaddleOCR; PaddleOCR(lang='vi', use_gpu=False, show_log=False)"`
       g. Generate test image: `python3 tests/fixtures/ocr-smoke-generator.py /tmp/ocr-smoke.jpg`
       h. Run OCR smoke test:
          ```bash
          python3 scripts/ocr_table.py /tmp/ocr-smoke.jpg /tmp/ocr-output.json
          python3 -c "
          import json, sys
          data = json.load(open('/tmp/ocr-output.json'))
          assert len(data['lines']) > 0, f'OCR returned 0 lines from smoke image'
          print(f'OCR smoke passed: {len(data[\"lines\"])} lines extracted')
          "
          ```

       IMPORTANT: Cache path is `~/.paddlex` (PaddleOCR 3.x), NOT `~/.paddleocr` (2.x).
       IMPORTANT: Include `env: PADDLE_PDX_MODEL_SOURCE: BOS` at the job level to prevent remote model check after cache restore.
