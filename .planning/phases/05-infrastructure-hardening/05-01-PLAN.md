<objective>
Build a static JSON CDN layer that pre-generates the most common API response payloads at build time and serves them from Vercel's CDN edge. On DB timeout or Supabase cold start, API routes fall back to the static files so users never see a 503.

Purpose: Eliminate Supabase as a single point of failure for read-heavy endpoints. Common queries (university list, scores by tohop/year) served from CDN in <500ms without hitting the DB at all.
Output: `scripts/generate-static-json.ts`, `public/data/` directory with JSON snapshots, updated route handlers with static fallback.
</objective>

  <action>
Update three route handlers to fall back to static JSON when Supabase is unavailable (DB_TIMEOUT).

Pattern for each handler — replace the `DB_UNAVAILABLE` 503 response with a static file read:

```typescript
import { readFileSync } from 'fs';
import { join } from 'path';

// In the DB_TIMEOUT catch block:
try {
  const staticData = JSON.parse(
    readFileSync(join(process.cwd(), 'public/data/universities.json'), 'utf-8')
  );
  return Response.json(staticData, {
    headers: {
      'Cache-Control': 'public, s-maxage=86400, stale-while-revalidate=3600',
      'X-Served-By': 'static-fallback',
    },
  });
} catch {
  return errorResponse('DB_UNAVAILABLE', 'Service temporarily unavailable', 503, {
    'Retry-After': '30',
  });
}
```

Apply this pattern to:
1. `app/api/universities/route.ts` — reads `public/data/universities.json`
2. `app/api/scores/route.ts` — reads `public/data/scores-by-tohop.json` (if the scores endpoint takes a tohop param, filter the in-memory JSON by `params.get('tohop')` before returning)
3. `app/api/tohop/route.ts` — reads `public/data/tohop.json`

The `X-Served-By: static-fallback` header lets monitoring/tests detect when fallback is active.

Use relative imports for `fs` and `path` — they're Node.js builtins, no import alias needed.
Do NOT change the normal (non-timeout) code path in any handler.
Do NOT add fallback to `/api/recommend` — it is personalized and cannot be served from static cache.
