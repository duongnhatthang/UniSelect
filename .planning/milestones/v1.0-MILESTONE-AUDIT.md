### Operational Gap: INFRA-02

`generate-static` script is not chained into the `build` command in `package.json`. Vercel Build Command must be manually configured as:
```
npm run generate-static && next build --webpack
```

Without this, future redeploys serve stale Phase 7 snapshot data. Phase 5 SUMMARY documents this as a manual setup step. Phase 7 committed static files as a stopgap.
