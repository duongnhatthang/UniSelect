**lib/db/timeout.ts** — Promise.race timeout per RESEARCH.md Pattern 4:
```typescript
export function withTimeout<T>(promise: Promise<T>, ms: number): Promise<T> {
  const timeout = new Promise<never>((_, reject) =>
    setTimeout(() => reject(new Error('DB_TIMEOUT')), ms)
  );
  return Promise.race([promise, timeout]);
}
```

**Engine implementation (lib/recommend/engine.ts):**

Export a single `recommend` function:
```typescript
export function recommend(input: RecommendInput, rows: CutoffDataRow[]): RecommendResult[]
```

Algorithm steps:
1. Group rows by composite key `${university_id}|${major_id}`
2. For each group:
   a. Sort by year ascending
   b. Take last 3 years only
   c. Compute weighted average: weights = [1, 2, 3] for 3 years, [1, 2] for 2, [1] for 1. Formula: sum(parseFloat(score) * weight) / sum(weights)
   d. Classify tier using input.total_score vs weighted_cutoff:
      - dream: total_score >= weighted_cutoff + 3
      - practical: weighted_cutoff - 1 <= total_score <= weighted_cutoff + 2
      - safe: weighted_cutoff - 5 <= total_score <= weighted_cutoff - 2
      - below safe: EXCLUDE from results
   e. Compute trend from last 2 years: diff > 0.5 = 'rising', diff < -0.5 = 'falling', else 'stable'. If only 1 year, trend = 'stable'
   f. Set data_years_limited = years.length < 3
3. Apply suggested_top_15:
   a. Sort all results: practical first, then dream, then safe
   b. Within each tier: practical sorted by ascending abs(total_score - weighted_cutoff), dream by descending margin, safe by descending score
   c. Deprioritize su pham: within each tier, move entries where major_name_vi contains "Su pham" or "Giao duc" (case-insensitive, Vietnamese diacritics handled via toLowerCase) to end of that tier group
   d. Mark first 15 as suggested_top_15=true, rest as false
4. Return full array (all qualifying results, not just top 15)

IMPORTANT: Always call parseFloat(row.score) — the score field is a string from Postgres numeric type.
